# LogQbit 文档

LogQbit 是一个面向实验室实验流程的数据记录工具包，核心能力包括：

- 基于目录的实验数据组织。
- 使用 `metadata.json` 和 `const.yaml` 管理元数据与常量参数。
- 通过 `LogFolder` 连续写入结构化数据。
- 在需要时使用浏览器和实时绘图工具进行可视化。
